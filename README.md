# San Francisco Housing Market Analysis

## Overview

This project analyzes rental housing trends in San Francisco using historical rental listing data.

The goal is to understand how rental prices have changed over time, identify geographic pricing differences, and explore the factors that influence housing costs across the city.

The analysis combines data cleaning, exploratory data analysis, time-series visualization, and housing market insights using Python.

---

## Objectives

- Analyze long-term rental price trends in San Francisco.
- Compare rental prices across cities and neighborhoods.
- Examine differences in rent by property size.
- Identify housing market patterns and pricing behavior.
- Visualize market trends using data-driven insights.

---
